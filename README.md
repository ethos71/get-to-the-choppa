## What is this?

Get To The Choppa is a humorous yet functional WiFi monitoring tool that helps you escape WiFi problems. When your WiFi goes down and everything falls apart, it's time to get to the choppa (your backup connection, that is).

## Installation

```bash
# Clone the repository
git clone https://github.com/ethos71/get-to-the-choppa.git
cd get-to-the-choppa

# Install dependencies (if using Python version)
pip install -r requirements.txt

# Or for Node.js version
npm install
```

## Usage

```bash
# Python version
python wifi_monitor.py

# Node.js version
node wifi_monitor.js
```

## What It Does

When WiFi problems arise:
1. Monitors your connection
2. Provides diagnostic information
3. Suggests alternative solutions (mobile hotspot, ethernet, coffee shop WiFi)
4. Motivates you with classic action movie quotes
